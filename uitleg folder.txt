Patient_data
    Hier kan iemand een gezipt vcf en een pdf bestand in stoppen.
    
Metadata
    Hier wordt een tekst bestand gemaakt voor elke keer als iemand het programma gebruikt. Hierin staan de naam van de gegeven files, de start datum en tijd en de versie van dit programma.

Scripts
    Hier staan alle python scripts die worden aangeroepen.

snpEff
    Hier staat het snpEFF programma. Hier hoeft niks mee te gebeuren.

temp
    Hier komen alle bestanden die tijdens het maken van het script gemaakt worden. Dus csv output, uitgepakte vcf, etc.
    Deze folder wordt leeg gemaakt aan het begin van elke run zodat er niet mega veel data wordt opgeslagen na elke run.

bash_script.txt
    Dit moet het bash script worden wat alles aanroept.
